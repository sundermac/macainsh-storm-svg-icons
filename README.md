# SVG Icons - HW #1

The MMED-1055 HW#1 is a one-page web application for the ORBITZ brand and its different flavours. It focuses on interactive SVGs that represent flavours: apple, blueberry, orange, mango, peach, and watermelon. When you hover over an SVG, it will animate, and when pressed, a message will be logged to the console.
